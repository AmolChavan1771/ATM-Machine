ATM Machine (Java, JDBC, MySQL)
 Project Description

This project is a Basic ATM Machine System developed using Java, JDBC, and MySQL.
It simulates simple banking operations like balance checking, cash deposit, withdrawal and transaction management using Object-Oriented Programming (OOPs) principles.

Objective
To understand and apply:
Java OOPs concepts (Encapsulation, Inheritance, Abstraction, Polymorphism)
CRUD operations using JDBC
Database connectivity between Java and MySQL
Building a console-based banking simulation system

Features (CRUD Operations)
✅ Create – Add new bank accounts
✅ Read – View account details and balance
✅ Update – Deposit, Withdraw, Change PIN
✅ Delete – Close/Delete an existing account
